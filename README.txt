How to add your project to GIT from local to cloud
(create a repo in GitHub account)

1. git init
2. git add README.md
3. git commit -m "chore:first commit"
4. git branch -M main
5. git remote add origin https://github.com/amolpatil389/coreJava (create a repo in GitHub account, this will be yours git hub repo link)
6. git push  -u origin main
